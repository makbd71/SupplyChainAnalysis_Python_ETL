# Supply-chain-etl-demand-planning

End-to-end Supply Chain Analytics &amp; ETL project using Python (Pandas, Matplotlib). Focused on demand planning, inventory optimization, and supplier lead-time analysis to support data-driven decision-making in a manufacturing/semiconductor environment.

🎯 Project Objective

This project simulates the role of a Supply Planning Analyst in a manufacturing/semiconductor environment. The goal is to analyze supply-demand data, identify inventory risks, and support production and procurement decisions using data-driven insights.

🤖 AI-Assisted Development

This project leveraged AI tools such as ChatGPT and Google Gemini for debugging, workflow optimization, and code structuring. All outputs were validated and aligned with real-world supply chain decision-making.
